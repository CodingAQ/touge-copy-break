# 头歌考试复制粘贴破解
破解头歌考试界面的复制粘贴限制，恢复 Ctrl+C/V 和右键功能

使用前需安装**篡改猴/篡改猴测试版**

## 使用方法

### 1. 从 Edge / Chrome 的插件商店安装 Tampermonkey（篡改猴）

#### 篡改猴稳定版
- [Edge扩展 - 篡改猴](https://microsoftedge.microsoft.com/addons/detail/%E7%AF%A1%E6%94%B9%E7%8C%B4/iikmkjmpaadaobahmlepeloendndfphd)
- [Chrome应用商店 - 篡改猴](https://chromewebstore.google.com/detail/tampermonkey/dhdgffkkebhmkfjojejmpbldmpobfkfo)

#### 篡改猴测试版
- [Edge扩展 - 篡改猴测试版](https://microsoftedge.microsoft.com/addons/detail/%E7%AF%A1%E6%94%B9%E7%8C%B4%E6%B5%8B%E8%AF%95%E7%89%88/fcmfnpggmnlmfebfghbfnillijihnkoh)
- [Chrome应用商店 - 篡改猴测试版](https://chromewebstore.google.com/detail/tampermonkey-beta/gcalenpjmijncebpfijmoaglllgpjagf)

### 2.根据指示给予篡改猴浏览器权限
